getCalculatedQuality()

getCalculatedQuality(): integer

Get calculated quality.

If the "quality" option is a number, that number is returned. If mime type of source is something else than "image/jpeg", the "default-quality" option is returned If quality is "auto" and source is a jpeg image, it will be attempted to detect jpeg quality. In case of failure, the value of the "default-quality" option is returned. In case of success, the detected quality is returned, or the value of the "max-quality" if that is lower.

doActualConvert()

doActualConvert()

The actual conversion is be done by a concrete converter extending this class.

At the stage this method is called, the abstract converter has taken preparational steps.

  • It has created the destination folder (if neccesary)
  • It has checked the input (valid mime type)
  • It has set up an error handler, mostly in order to catch and log warnings during the doConvert fase

Note: This method is not meant to be called from the outside. Use the static convert method for converting or, if you wish, create an instance with ::createInstance() and then call ::doConvert()
