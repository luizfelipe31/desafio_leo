setProvidedOptions()

setProvidedOptions(array  $providedOptions = array()): void

Set "provided options" (options provided by the user when calling convert().

This also calculates the protected options array, by merging in the default options, merging jpeg and png options and merging prefixed options (such as 'vips-quality'). The resulting options array are set in the protected property $this->options and can be retrieved using the public ::getOptions() function.

Parameters

array $providedOptions

(optional)

Note that the defaults depends on the mime type of the source. For example, the default value for quality is "auto" for jpegs, and 85 for pngs.
