checkAbsolutePath()

checkAbsolutePath(string  $absFilePath,  $text = 'file'): void

Check absolute file path to prevent attacks.

  • Prevents non printable characters
  • Prevents stream wrappers
  • Prevents directory traversal

Preventing non printable characters is especially done to prevent the NUL character, which can be used to bypass other tests. See https://st-g.de/2011/04/doing-filename-checks-securely-in-PHP.

Preventeng stream wrappers is especially done to protect against Phar Deserialization. See https://blog.ripstech.com/2018/new-php-exploitation-technique/
